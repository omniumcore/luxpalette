## 🇺🇸 English Version

Welcome to **LuxPalette**, a premium, open-source web tool designed for designers, indie developers, and content creators! This application allows you to explore the color wheel, calculate harmonic tones in triad mode, and instantly browse a massive library of 500 color palettes categorized by style.

### 🚀 Key Features
* **🎨 Interactive Color Wheel:** Explore shades freely or toggle **Triad Mode** to automatically calculate three harmonic colors in a perfect 120-degree equilateral split.
* **⚡ 500 Procedural Palettes:** Ultra-fast navigation optimized across 14 independent categories (Pastel, Neon, Metallic, Red, Blue, etc.).
* **🔍 Real-Time Search:** Find colors instantly by searching for tone names, categories, or direct Hexadecimal codes.
* **🌐 Language Toggle:** Switch the entire interface between Spanish and English with a single click in milliseconds.
* **📋 Smart Clipboard:** Click directly on any palette card's data row to cleanly copy only the value you need (`HEX`, `RGB`, or `HSV`).
* **💳 Integrated Support Window:** Clean, non-intrusive modal pop-up to receive global and local voluntary donations.

### 🛠️ Tech Stack
* **HTML5 / JSX:** Semantic application structure and interactive modal layout.
* **CSS3 (Tailwind CSS):** Responsive dark mode design, premium background blur effects (`backdrop-filter: blur`), and hover scaling animations.
* **JavaScript (React / TypeScript):** Mathematical logic for radial wheel calculations, dynamic procedural palette generation, and clipboard handling.

## 🇲🇽 Versión en Español

¡Bienvenido a **LuxPalette**, una herramienta web premium y de código abierto diseñada especialmente para diseñadores, programadores independientes y creadores de contenido! Esta aplicación te permite explorar el círculo cromático, calcular armonías en modo tríada y navegar de forma instantánea por una biblioteca masiva de 500 paletas de colores divididas por categorías.

### 🚀 Características Principales
* **🎨 Rueda de Color Interactiva:** Explora tonalidades de forma libre o activa el **Modo Tríada** para calcular tres colores armónicos automáticos en un triángulo perfecto de 120°.
* **⚡ 500 Paletas Procedurales:** Navegación ultra veloz optimizada a través de 14 categorías independientes (Pastel, Neón, Metálicos, Rojos, Azules, etc.).
* **🔍 Buscador en Tiempo Real:** Encuentra colores al instante buscando por nombre de tono, categoría o código Hexadecimal directo.
* **🌐 Selector de Idiomas:** Cambia toda la interfaz entre Español e Inglés con un solo clic en milisegundos.
* **📋 Copiado Inteligente:** Haz clic directamente sobre la línea de datos de cualquier tarjeta para copiar de forma aislada únicamente el valor que necesitas (`HEX`, `RGB` o `HSV`).
* **💳 Ventana de Soporte Integrada:** Interfaz limpia y no invasiva para recibir donaciones voluntarias globales y locales.

### 🛠️ Tecnologías Utilizadas
* **HTML5 / JSX:** Estructura semántica de la aplicación y maquetación del modal interactivo.
* **CSS3 (Tailwind CSS):** Diseño responsivo en modo oscuro, efectos premium de difuminado de fondo (`backdrop-filter: blur`) y animaciones de expansión al pasar el cursor (`hover scaling`).
* **JavaScript (React / TypeScript):** Lógica matemática para el cálculo radial de la rueda en modo tríada, algoritmos para la generación procedural de las 500 paletas en tiempo real y gestión del portapapeles.
